# NutriFit Pro

NutriFit Pro é uma aplicação para gestão nutricional com painel administrativo, cadastro de clientes, consultas, metas e medidas corporais.

## Funcionalidades

- Login do nutricionista
- Cadastro e gestão de clientes
- Agendamento de consultas
- Registro de metas
- Acompanhamento de medidas corporais
- Interface com tema claro e escuro

## Tecnologias

- React
- Vite
- Node.js
- Express
- SQLite

## Estrutura do projeto

- frontend: interface web em React
- backend: API em Express
- backend/db: banco SQLite e inicialização do schema

## Como executar

### Backend

```bash
cd backend
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

A aplicação pode ser acessada no navegador através do frontend em desenvolvimento.
